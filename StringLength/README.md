# 文本长度和重复字串分析器

这个项目包含两个Python脚本，用于分析文本的长度和检测重复的字符串。

## 文件说明

### 1. `simple_text_analyzer.py` - 简单版本
- 统计文本字符长度
- 检测重复的字串及其出现次数
- 简洁的输出格式

### 2. `text_analyzer.py` - 完整版本
- 详细的文本统计信息（字符数、单词数、行数等）
- 字符类型分析（字母、数字、空格、标点符号）
- 智能重复字串检测
- 美观的输出格式

## 使用方法

### 运行简单版本：
```bash
python simple_text_analyzer.py
```

### 运行完整版本：
```bash
python text_analyzer.py
```

## 功能特点

- **字符长度统计**: 准确计算输入文本的字符数量
- **重复字串检测**: 自动识别文本中重复出现的子字符串
- **重复次数统计**: 显示每个重复字串的出现次数
- **交互式界面**: 支持连续输入多个文本进行分析
- **中文支持**: 完全支持中文文本分析

## 示例

输入文本: "hello world hello"
输出:
- 字符长度: 17
- 重复字串: "hello" - 重复 2 次

## 退出程序

输入 `quit` 或 `exit` 即可退出程序。
